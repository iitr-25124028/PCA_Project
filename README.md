# PCA on Gene Expression Data

This project performs Principal Component Analysis (PCA) on breast cancer gene expression data from GEO dataset GSE5325.

## Objectives

1. Extract XBP1 and GATA3 expression values
2. Generate scatter plot
3. Perform PCA
4. Visualize patient clustering

## Files

- pca_analysis.py
- class.tsv
- columns.tsv.gz
- filtered.tsv.gz

## Output

- scatter_plot.png
- pca_plot.png
